## Features Supported

Our custom parser supports the following:

- **Headers** (H1 to H6)
- **Bold text** using **double asterisks** or __double underscores__
- *Italic text* using *single asterisks* or _single underscores_
- [Links](https://example.com) like this one
- `Inline code` formatting
- Unordered lists like this one
